# MergeCraft - A Python PDF Merger

MergeCraft is a lightweight Python script designed for one thing and one thing only: merging PDFs swiftly and efficiently. Whether you're piecing together scattered documents or combining reports, MergeCraft has got your back. 

---

## Description

Ever found yourself drowning in a sea of PDF files and just wished there was a quick way to merge them? MergeCraft is your solution. It’s simple, fast, and doesn’t require fancy software—just Python and a few dependencies. Whether you’re a student, office worker, or developer, MergeCraft streamlines your PDF merging needs. 

---

## Features

- **Swift PDF Merging**: Combine multiple PDFs into one file in seconds.  
- **User-Friendly**: Straightforward interface—no unnecessary frills.  
- **Lightweight**: No bulky software or tools required, runs directly from the terminal.  
- **Customizable**: Tailor the script to meet your specific PDF merging requirements.  

---

## Installation and Usage

Getting started with MergeCraft is quick and painless. Follow these steps:

1. **Clone the Repository**  
   ```bash
   git clone https://github.com/manik765444-ai/mergecraft.git
   cd mergecraft
   ```

2. **Install Dependencies**  
   MergeCraft requires a few Python libraries to work its magic. Install them using:  
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Script**  
   You’re now ready to merge your PDFs! Execute the script using:  
   ```bash
   python main.py
   ```

4. **Follow the Prompts**  
   The script will guide you through selecting and merging your PDFs effortlessly.
